# Медосмотр в Анталье для SGK

Пошаговая инструкция по записи и прохождению медосмотра в Анталье с учетом требований SGK.

## Важный факт

Для целей SGK принимается медосмотр **только** из `Antalya Eğitim ve Araştırma Hastanesi` (в исходном тексте: Antalya ve Egitim Hastanesi). В других больницах может быть ближе или проще поймать слот, но офис SGK их не принимает.
